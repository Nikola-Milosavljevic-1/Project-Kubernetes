# Project-Kubernetes

Ce projet initialise une application composée d’un **frontend**, d’un **backend**
et d’une **base de données MongoDB**.

L’objectif est de fournir une base technique simple
afin de permettre le développement ultérieur des fonctionnalités.

---
Chaque composant est déployé avec :
- un **Deployment**
- un **Service**

 Frontend : application web simple
- Backend : API Node.js
- Database : MongoDB
- Orchestration : Kubernetes

---

## Structure du projet
Kubernetes gère :
- le déploiement des conteneurs
- la communication entre services
- la base pour la montée en charge

---

## Objectif du dépôt

Ce dépôt se concentre sur l’**initialisation** du projet.
Les développements fonctionnels seront réalisés ultérieurement.
